# EasyVisa
(Boosting, Bagging, Model Tuning)Help hiring companies identify candidates with the best approval rate to be certified for a visa. I experimented with tree-based models (Decision Tree, Random Forest), and Boosting methods (Adaboost, XGBoost, Gradient Boosting). The tuned gradient boosting model gave 75% accuracy and provided an 82.6% F1 score.
